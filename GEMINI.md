# Siripong Rojchaiwiriyakul Portfolio - "Executive Gold" Edition

This directory contains the premium, executive-tier portfolio of Siripong Rojchaiwiriyakul, a Senior Sales Analyst and Data Strategist.

## Project Overview

*   **Aesthetic:** "Executive Gold" design system featuring deep midnight backgrounds, metallic gold accents, and sophisticated typography.
*   **Purpose:** A high-impact digital presence that reflects 10+ years of corporate leadership and a significant personal brand (Book Intelligent).
*   **Tech Stack:** 
    *   **HTML5/CSS3:** Advanced layouts using CSS Grid and Flexbox.
    *   **Typography:** `Inter` (Sans-serif) for an elegant feel and `JetBrains Mono` for technical/data context.
    *   **Visuals:** Subtle background glows, gold-outlined cards, and refined transition animations.

## Key Features

*   **Premium Branding:** The site uses a custom "Executive Gold" palette to evoke authority and trust.
*   **Strategic Timeline:** A luxurious vertical timeline detailing corporate achievements at Ford and ExxonMobil.
*   **Impact-Focused Projects:** Card layouts that highlight quantifiable results (+41% sales, 3-hour workload reduction).
*   **Book Intelligent Hub:** A dedicated premium section for the education brand with metallic stat counters.
*   **Glassmorphism & Depth:** Use of semi-transparent surfaces and soft shadows to create a multi-layered, modern UI.

## Development Conventions

*   **Design System:** Use CSS variables defined in `:root` for consistency.
*   **Spacing:** Prioritize significant padding (8rem+ for sections) to maintain an "Executive" feel with ample negative space.
*   **Accents:** Use the `--gold` variable sparingly for high-impact elements like headers, icons, and primary buttons.
*   **Responsive:** Fully optimized for all viewports, ensuring the premium feel is maintained on mobile devices.
